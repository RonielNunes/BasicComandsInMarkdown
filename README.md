# Basic Comands In Markdown

# Títulos:

```Markdown
# Títulos 1
## Títulos 2
### Títulos 3
#### Títulos 4
##### Títulos 5
###### Títulos 6
```
Visualizando 
# Títulos 1
## Títulos 2
### Títulos 3
#### Títulos 4
##### Títulos 5
###### Títulos 6


# Texto em Negrito
```Markdown
**Negrito**
```
Visualizando: **Texto em negrio**.

# Texto em itálico

```

# Tabela

```Markdown
nome   | idade
:----: | :----:
A      | 12
B      | 13
```

Visualizando:

nome   | idade
:----: | :----:
A      | 12
B      | 13
 

# Código

```Markdown
  <```><nome da linguagem>
  //code
  <```>
```

Exemplo:

```C
int main(){
    return 0;
}
```

# Código em partes textuais

```Markdown
` code `
```
Exemplo: para retorna basta colocar `return 0;`
