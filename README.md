* c++

* python

# **Código**

```Markdown
  <```><nome da linguagem>
  //code
  <```>
```

Exemplo:

```C
int main(){
    return 0;
}
```

# **Código em partes textuais**

```Markdown
` code `
```
Exemplo: para retorna basta colocar `return 0;`
