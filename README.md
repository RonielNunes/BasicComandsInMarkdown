# Negrito
```Markdown
**Negrito**
```
Visualizando: **Texto em negrio**.

```Markdown
*itálico* ou _itálico_
```

```Markdown
* marcados em lista
* java
* css
* c++
* c
* python
```
  
```Markdown
- Outra forma de marcar
- java
- R
- html
```

```Markdown
1. Outra forma de marcar
2. Ruby
3. SQL
```

# Código

```Markdown
  ```<nome da linguagem>
  //code
  ```
```
Exemplo

```C
int main(){
    return 0;
}
```

# Código em partes textuais

```Markdown
` code `
```
Exemplo: para retorna basta colocar `return 0;`
